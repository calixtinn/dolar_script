Descrição:
==========

---------------------------------------------------------------------------------------------------------------------------------
Este é um script utilizando linguagem bash e awk, que ao ser executado, mostra no terminal a atual cotação do Dólar comercial americano em Reais (R$). Gera também um gráfico contendo a variação do valor do Dólar nos últimos 30 dias, caso o usuário decida
instalar o software GNUPlot para a plotagem do mesmo. Se o usuário não quiser instalar o GNUPlot, o gráfico não será gerado, e
o script mostrará na tela apenas o valor da cotação atual do Dólar e dos valores máximo e mínimo durantes esses últimos 30 dias.
---------------------------------------------------------------------------------------------------------------------------------

Utilização: 
===========

1 - Dar permissão de execução para o script utilizando o comando: chmod +x /<diretorio>/dolar_script

2 - Executar no terminal digitando: ./dolar_script
